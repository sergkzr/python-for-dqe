- hw1-hw5: 
    - cd to root directory of repository
    - run 'python hwx.py' command, x in (1, 2, 3, 4, 5)
- hw6-10:
    - cd to hwx directory
    - run 'python app.py' command, x in (6, 7, 8, 9, 10)
